# HDC Dashboard Technical Documentation

เอกสารนี้สรุปเงื่อนไข ลอจิกการทำงาน และโครงสร้างการจัดวางหน้าจอ (Layout Layer) ของหน้า `src/app/dashboard/hdc/page.tsx`

## 1. เงื่อนไขและพฤติกรรมของตัวกรอง (Filters)

ระบบตัวกรองถูกออกแบบให้มีความสัมพันธ์กันแบบ Cascading (บนลงล่าง) เพื่อความแม่นยำในการเลือกพื้นที่:

### 1.1 ตัวกรองวันที่ (จากเดือน - ถึงเดือน)
- **ข้อมูลตัวเลือก**: แสดงรายการเดือนทั้งหมดที่พบในฐานข้อมูล (Parquet) เรียงจากใหม่ไปเก่า
- **ค่าเริ่มต้น (Default)**: 
  - `ถึงเดือน` (endDate): เลือกเดือนล่าสุดที่มีข้อมูล
  - `จากเดือน` (startDate): คำนวณย้อนกลับไปยังวันที่ `1 ตุลาคม` ของปีงบประมาณนั้นๆ (ถ้าเดือนล่าสุด >= ตุลาคม ใช้ปีนั้น, ถ้า < ตุลาคม ใช้ปีก่อนหน้า)

### 1.2 ตัวกรองพื้นที่ (เขต -> จังหวัด -> อำเภอ -> ตำบล)
- **เขตสุขภาพ (Regions)**: เปลี่ยนเขตจะล้างค่า จังหวัด, อำเภอ และตำบล ทั้งหมด
- **จังหวัด (Provinces)**: แสดงเฉพาะจังหวัดในเขตที่เลือก, เปลี่ยนจังหวัดจะล้างค่า อำเภอ และตำบล
- **อำเภอ/เขต (Districts)**: แสดงเฉพาะอำเภอในจังหวัดที่เลือก, เปลี่ยนอำเภอจะล้างค่าตำบล
- **ตำบล/แขวง (Subdistricts)**: แสดงเฉพาะตำบลในอำเภอที่เลือก

### 1.3 ตัวกรองทางคลินิก
- **กลุ่มโรค (Disease Groups)**: สามารถเลือกได้หลายกลุ่มโรค (Multi-select) เพื่อดูสถิติรวม
- **การวินิจฉัย (Diagnosis Types)**: เลือกได้เพียง 1 ประเภท (Single-select) เพื่อกรองความเข้มข้นของการวินิจฉัย (เช่น เฉพาะโรคหลัก ร่วมกับ Z58.1)

## 2. โครงสร้างการแสดงผลข้อมูล (Data Visualization)

### 2.1 ส่วนสรุปสถิติ (Summary Cards)
- **จำนวนผู้ป่วยทั้งหมด**: นับจำนวนผู้ป่วยสะสม (Unique Patients) โดยอิงจากการวินิจฉัยทั้งหมดในฐานข้อมูล
- **จำนวนการวินิจฉัยทั้งหมด**: นับจำนวนครั้งที่เข้ามารับบริการตามเงื่อนไขตัวกรอง
- **Top 5 Disease Stats**: การ์ดขนาดเล็ก 5 ใบ แสดงจำนวนผู้ป่วยแยกตามกลุ่มโรค พร้อมแถบสีประจำโรค

### 2.2 กราฟแนวโน้มรายเดือน (Monthly Trend Chart)
- **ประเภท**: กราฟแท่งซ้อน (Stacked Bar Chart) แสดงจำนวนผู้ป่วยแยกตามโรค + กราฟเส้น (Line Chart) แสดงค่าเฉลี่ย PM2.5
- **แกน Y (Dual Axis)**:
  - **ซ้าย (สีขาว)**: จำนวนผู้ป่วย (ราย) - คำนวณ Max อัตโนมัติจากข้อมูลในจอ + 10%
  - **ขวา (สีแดง)**: ค่าเฉลี่ยฝุ่น PM2.5 (µg/m³) - คำนวณ Max อัตโนมัติ + 10%
- **Interactive Tooltip**: เมื่อ Hover ที่แท่งกราฟ จะปรากฏหน้าต่างรายละเอียดขนาดใหญ่ แสดงสถิติโรคแยกย่อยและค่าฝุ่นในเดือนนั้นๆ

### 2.3 แผนที่รายงาน (Thailand Map)
- **ข้อมูลที่แสดง**: จำนวนผู้ป่วยรายจังหวัด และ "อัตราป่วยต่อแสนประชากร" (Rate per 100,000)
- **การคำนวณอัตราป่วย**: นำจำนวนผู้ป่วย หารด้วยประชากรกลางปีจากไฟล์ `mid_year.csv`
- **ระดับสี**: แบ่งเป็น 5 ระดับ (เขียว, ฟ้า, เหลือง, ส้ม, แดง) ตามความหนาแน่นของผู้ป่วย

## 3. โครงสร้างเลเยอร์และการจัดตำแหน่ง (Layout & Positioning)

### 3.1 Global Structure
- **พื้นหลัง**: รูปภาพ `background.jpg` (Fixed & Cover) ซ้อนทับด้วย Overlay สีเข้ม `bg-slate-900/40`
- **Container**: `max-w-screen-2xl` จัดวางแบบ `flex flex-col gap-4`

### 3.2 Positioning Layers (z-index)
- **Header (`z-[60]`)**: บาร์ด้านบนแสดงโลโก้ DDC/HDC และสถานะสิทธิ์ของผู้ใช้
- **Filters (`z-[50]`)**: แถบตัวกรอง Glassmorphism 8 คอลัมน์ (ในจอใหญ่) วางอยู่บนสุดเพื่อให้ Dropdown ไม่ถูกทับ
- **Summary Cards (`z-[30]`)**: ส่วนแสดงตัวเลขสถิติหลัก
- **Content Layer (`z-[20]`)**: 
  - **ซ้าย (1.6fr)**: กราฟแนวโน้มรายเดือน
  - **ขวา (1.1fr)**: แผนที่ประเทศไทย

### 3.3 การจัดการพื้นที่บนมือถือ (Responsive)
- ตัวกรองจะยุบลงมาเป็นแถวแนวตั้ง
- สถิติกลุ่มโรค (Bottom Row) จะเปลี่ยนจาก 5 คอลัมน์ เป็น 2-3 คอลัมน์ตามความกว้างหน้าจอ
- กราฟและแผนที่จะแสดงเรียงต่อกันในแนวตั้ง
